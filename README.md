# StudyNest 📚

**StudyNest** is a modern student resource website designed to help learners easily find **programming roadmaps, study tips, learning resources, and useful websites** in one place.

The goal of this project is to solve a common problem: students often struggle to find **organized notes, tools, and learning paths**. StudyNest brings everything together in a clean and structured platform.
